If all prerequisites pass, run the migration from the `apps/api` directory:

```bash
cd apps/api && npx typeorm migration:run -d src/data-source.ts
```

If the command fails because `ts-node` cannot transpile on the fly, try:
```bash
cd apps/api && npx ts-node -r tsconfig-paths/register -e "require('typeorm/cli').run()" migration:run -d src/data-source.ts
```

Or use the project's npm script if one exists (check `apps/api/package.json` for a `migration:run` script).

After the migration runs, verify the schema changes in the database:
