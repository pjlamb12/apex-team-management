<task type="auto">
  <name>Task 1: Implement getSeasonStats in SeasonsService</name>
  <files>apps/api/src/teams/seasons.service.ts</files>
  <action>
    - Add `getSeasonStats(teamId: string, seasonId: string)` to `SeasonsService`.
    - Business logic:
      - Verify season belongs to team.
      - Fetch all `completed` events of type `game` for the `seasonId`.
      - Iterate through games and sum `goalsFor` and `goalsAgainst`.
      - Calculate Wins, Losses, Draws based on scores.
      - Win: `goalsFor > goalsAgainst`, Loss: `goalsFor < goalsAgainst`, Draw: `goalsFor === goalsAgainst`.
      - Exclude games where either score is `null` from W/L/D counts.
      - Return `SeasonStats` object: `{ wins, losses, draws, goalsFor, goalsAgainst, goalDifference }`.
  </action>
  <verify>
    <automated>nx test api --testFile=apps/api/src/teams/seasons.service.spec.ts</automated>
  </verify>
  <done>Business logic correctly aggregates season performance from game results.</done>
