  <action>
    - Update `TeamsService`:
      - `findAllByCoach(userId)`: Update to search via `members` relation (finding teams where user is any role) AND `coachId` (for legacy/fallback).
      - `create`: Update to create a `TeamMemberEntity` with `HEAD_COACH` role for the creator. Also generate an initial `joinCode`.

      - Add `regenerateCode(teamId)`: Generate new code, update team, ensure uniqueness in DB.
    - Update `TeamsController`:
      - Add `POST /teams/join` with `{ code }`.
      - Add `POST /teams/:id/code/regenerate`.

  </action>
