# Plan 07-02: Native Platform Configuration

**Phase:** 07-pwa-native-builds
**Plan:** 02
**Goal:** Add Android and iOS native platforms to the Capacitor project.

## Requirements
- **INFR-05:** Capacitor configured for Android.
- **INFR-06:** Capacitor configured for iOS.

## Context
- Capacitor CLI and core installed (07-01).
- `capacitor.config.ts` initialized.
- Frontend build output available at `dist/apps/frontend/browser`.

## Tasks
1. **Add Android Platform:** Execute `npx cap add android`.
2. **Add iOS Platform:** Execute `npx cap add ios`.
3. **Initial Sync:** Run `npx cap sync` to ensure initial web assets are copied.

## Implementation Strategy
- Run the `cap add` commands.
- Verify directories `android/` and `ios/` are created in the project root.

## Verification Plan
- Check for `android/build.gradle`.
- Check for `ios/App/App.xcodeproj`.
