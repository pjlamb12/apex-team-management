<objective>
Remove all hardcoded black values from the game console components. These four files contain per-component hardcodes that override (or bypass) the global AP token system established in Plan 01.

Purpose: The global token overrides from Plan 01 cannot reach values that are hardcoded directly in component SCSS or HTML templates. The console-wrapper has `--background: #000000` on `ion-toolbar` that explicitly wins over the global `--ion-toolbar-background`. The event-log has `#ffffff` fallbacks that flash white on load. These require surgical per-file fixes.

Output:
- console-wrapper.scss — `--background: #000000` removed, `#111111` replaced with `var(--ap-surface-raised, #263044)`, `font-weight: 800` corrected to `700`
- console-wrapper.html — `bg-gray-950` → `bg-ap-bg`, both `bg-gray-900` → `bg-ap-surface-raised`
- event-log.scss — all four `#ffffff`/light-gray fallbacks replaced with dark alternatives
- soccer-pitch-view.html — `bg-black/60` → `bg-slate-900/80` on player name label
</objective>
