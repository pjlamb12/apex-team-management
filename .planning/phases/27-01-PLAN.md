## Technical Details

### Import Schema (Strict)
```json
{
  "name": "string (required)",
  "description": "string (required)",
  "sourceUrl": "string (optional)",
  "instructions": [
    { "step": 1, "text": "..." },
    { "step": 2, "text": "..." }
  ],
  "tagNames": ["string", "string"]
}
```

### Constraints
- Must use existing `findOrCreateTags` to avoid duplicate tags.
- Must ensure `coachId` is correctly assigned from the authenticated user.
