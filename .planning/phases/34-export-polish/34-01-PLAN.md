  <name>Task 2: Implement CSV Export Service and Controller</name>
  <files>apps/api/src/analytics/export/csv-export.service.ts, apps/api/src/analytics/analytics.controller.ts, apps/api/src/analytics/analytics.module.ts</files>
  <action>
    - Create `CsvExportService` in `apps/api/src/analytics/export/csv-export.service.ts`.
    - Implement `generate(teamId: string, options: ExportOptionsDto)`:
        - Fetch data from `PerformanceMetricsService` and `PlayingTimeService`.
        - If `granularity` is 'per-game', fetch stats per event.
        - Map stats into a flat array of objects for `json2csv`.
        - Headers should include: Player Name, Goals, Assists, Yellow Cards, Red Cards, Games Played, Total Minutes.
    - Update `AnalyticsController`:
        - Add `@Get('analytics/export/csv')` endpoint.
        - Use `@Res() res: Response` from `express` to stream the CSV file.
        - Set headers: `Content-Type: text/csv`, `Content-Disposition: attachment; filename="team-analytics.csv"`.
    - Register `CsvExportService` in `AnalyticsModule`.
  </action>
  <verify>
    <automated>npx nx test api --testFile=export</automated>
  </verify>
  <done>CSV export endpoint is functional and returns correctly formatted data.</done>
