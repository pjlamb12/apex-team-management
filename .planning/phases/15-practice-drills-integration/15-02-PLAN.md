  <behavior>
    - `findAllForEvent(userId, eventId)`: Returns drills for event if coach owns the event (via season.team). Ordered by sequence.
    - `addDrillToPlan(userId, eventId, dto)`:
        - Verifies event exists and type is 'practice'.
        - Verifies userId owns Event (event.season.team.coachId === userId).
        - Verifies userId owns Drill (drill.coachId === userId).
        - Calculates sequence (max + 1).
        - Saves and returns PracticeDrillEntity.
    - `update(userId, eventId, practiceDrillId, dto)`:
        - Verifies userId owns Event and association.
        - If `teamRating` is provided, verifies `event.status` is 'in_progress' or 'completed'.
        - Updates fields and saves.
    - `remove(userId, eventId, practiceDrillId)`:
        - Verifies ownership and removes association.
    - `reorder(userId, eventId, dto)`:
        - Verifies ownership of Event.
        - Updates all sequences based on `dto.ids` in a transaction.
  </behavior>
