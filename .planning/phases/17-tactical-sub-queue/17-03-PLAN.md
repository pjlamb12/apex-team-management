  <files>libs/client/feature/game-console/src/lib/console-wrapper/console-wrapper.ts, libs/client/feature/game-console/src/lib/console-wrapper/console-wrapper.html</files>
  <action>
    - Import and register `SubQueueComponent` in `ConsoleWrapper`.
    - Update `handlePlayerSelection`: 
        - When a bench player is selected and a pitch player is tapped, call `stateService.stageSub(inPlayerId, outPlayerId)`.
        - Do NOT push a 'SUB' event immediately.
    - Implement `handleApplySubs()`:
        - Capture `currentMinute` from `clockService.currentMinute()` once.
        - Capture `timestamp = Date.now()` once.
        - Get `stagedSubs()` and current `activePlayers()`.
        - Map `stagedSubs` to `GameEvent[]` array:
            - Find `outPlayer` in `activePlayers` to get their current `slotIndex`.
            - Set `type: 'SUB'`, `playerIdIn`, `playerIdOut`, `slotIndex`, `minuteOccurred: currentMinute`, `timestamp`.
        - Call `stateService.pushEvents(batch)`.
        - Call `stateService.clearStagedSubs()`.
    - Implement `handleUnstageSub(playerId)` and `handleClearSubs()`.
    - Update template to pass signals to `BenchView`, `SoccerPitchView`, and the new `SubQueueComponent`.
  </action>
  <verify>

  </verify>
