# Phase 36: Scouting & Evaluation

## Goal
Enable coaches to perform structured evaluations of candidates using configurable rubrics and performance ratings.

## Requirements
- **TRY-EVAL-01**: Define a `ScoutingRubric` entity that allows teams to configure their evaluation categories (e.g., Technical, Tactical, Athleticism).
- **TRY-EVAL-02**: Implement a `CandidateEvaluation` entity to store specific ratings (1-5 stars) and qualitative notes for each candidate per session or overall.
- **TRY-EVAL-03**: Support "Overall Tryout Notes" on the tryout event itself (already supported by `EventEntity.notes`).

## Proposed Data Schema

### ScoutingRubricEntity
- `id`: UUID
- `teamId`: string (FK to TeamEntity)
- `name`: string (e.g., 'Technical skills', 'Attitude', 'Tactical awareness')
- `description`: text (nullable)
- `weight`: integer (default 1) - For future use in weighted averages

### CandidateEvaluationEntity
- `id`: UUID
- `candidateId`: string (FK to CandidateEntity)
- `eventId`: string (FK to EventEntity, nullable for "overall" evaluations)
- `rubricId`: string (FK to ScoutingRubricEntity)
- `rating`: integer (1-5)
- `notes`: text (nullable)
- `coachId`: string (FK to UserEntity)

## Implementation Tasks

### 1. Backend (NestJS)
- [ ] Create `ScoutingRubric` and `CandidateEvaluation` entities.
- [ ] Implement `ScoutingService` and `ScoutingController`.
- [ ] Implement CRUD for rubrics per team.
- [ ] Implement evaluation entry and retrieval (per candidate, per event).

### 2. Frontend (Angular)
- [ ] Define `ScoutingRubric` and `CandidateEvaluation` interfaces.
- [ ] Create `ScoutingService` in data-access library.
- [ ] Build a "Rubric Management" view for coaches to set up their evaluation criteria.
- [ ] Build an "Evaluation Form" component for use during tryout sessions.

## Success Criteria
1. Coaches can create custom scouting rubrics for their team.
2. Coaches can record 1-5 star ratings for candidates against specific rubrics.
3. Evaluations can be retrieved per candidate to show a performance summary.
